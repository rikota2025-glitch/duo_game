# Design: タイトル画面 & まとめて水を移す機能

Date: 2026-06-21

## 概要

**Water Water Color** に2つの機能を追加する。

1. **タイトル画面**: ゲーム起動時に最初に表示するシンプルな画面
2. **まとめて移動**: 上から同じ色が連続している液体を一度にまとめて移す

実装方針: 最小変更（Aプラン）。既存の4ファイル構成を維持し、TDDで進める。

---

## 画面フロー

```
タイトル画面 → レベル選択画面 → ゲーム画面
```

- 変更前: レベル選択 → ゲーム
- 変更後: タイトル → レベル選択 → ゲーム
- タイトル画面はゲームタイトルと「スタート」ボタンのみ
- ボタン押下で既存の `showScreen('level-select-screen')` を呼ぶ

---

## まとめて移動の仕様

### 新関数: `countMovable(tubes, fromIndex)`

- 試験管の上から連続して同じ色の液体が何個あるかを返す
- 例: `["red", "blue", "blue"]` → `2`（上2個が `"blue"`）
- 空の試験管の場合は `0` を返す

### `executeMove` の変更

- `countMovable` で移せる個数を取得
- 移動先の空きスペース: `TUBE_CAPACITY - to.length`
- 実際に移動する個数: `Math.min(countMovable結果, 空きスペース)`
- 該当個数をまとめて `from` からポップし `to` にプッシュ
- カウント(`moves`)は何個移動しても `+1`

### `canMove` は変更なし

- 移動元の一番上の色と移動先の一番上の色の互換チェックはそのまま使える

## テスト計画（TDD）

### ユニットテスト (`game.test.js`)

**`countMovable`:**
- 上から同じ色が3個連続 → 3
- 上1個だけ同じ色 → 1
- 試験管が空 → 0
- 全部同じ色 → 全個数

**`executeMove`（まとめ移動）:**
- 同じ色2個が連続 → 2個まとめて移動、moves +1
- 移動先の空きが1しかない → 1個だけ移動
- Undoで元に戻る

### 統合テスト (`game.test.html`)

- タイトル画面が最初に表示される
- スタートボタンでレベル選択画面に遷移
- レベル選択 → ゲーム画面は既存フロー通り
- まとめ移動を含む一連のゲームプレイでクリア判定が正しく動く
- 既存テストのリグレッション確認
